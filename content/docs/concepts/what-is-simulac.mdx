# What Simulac Is

Simulac is a Python package for running remote robotics benchmarks today and for building local environments through an evolving world-building API.

Simulac's goal is to provide a unified testing environment for physical AI development through benchmarks, along with a unified API for building environments.

More benchmarks will continue to be added beyond the ones available today. The environment-building API is currently limited to `mujoco`, but it is intended to expand over time to include `newton` and `genesis` through the same interface.

Long term, Simulac aims to become a developer-centric digital twin builder.
